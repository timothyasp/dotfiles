Be aware that this imports</div><div class='line' id='LC22'>&nbsp;&nbsp;&nbsp;modules to perform introspection and assumes that importing a module does</div><div class='line' id='LC23'>&nbsp;&nbsp;&nbsp;not have serious side effects (although it might, however it shouldn&#39;t).</div><div class='line' id='LC24'><br/></div><div class='line' id='LC25'>&nbsp;- You can enable automatic variable completion after typing a dot or if you</div><div class='line' id='LC26'>&nbsp;&nbsp;&nbsp;type a space after &#39;import&#39; on a &#39;from &lt;module&gt; import &lt;variable&gt;&#39; line.</div><div class='line' id='LC27'>&nbsp;&nbsp;&nbsp;(this is not enabled by default because of the side effect issue mentioned</div><div class='line' id='LC28'>&nbsp;&nbsp;&nbsp;above).</div><div class='line' id='LC29'><br/></div><div class='line' id='LC30'>Experimental features:</div><div class='line' id='LC31'><br/></div><div class='line' id='LC32'>&nbsp;- The plug-in now comes with a Python script that uses the AST module [2] in</div><div class='line' id='LC33'>&nbsp;&nbsp;&nbsp;the Python standard library to implement a type inference engine which can</div><div class='line' id='LC34'>&nbsp;&nbsp;&nbsp;be used to suggest completion candidates.
